<h1 align="center"><b>Pixel-Monster-Bestiary</b></h1>

An online bestiary for displaying the art, sounds, and descriptions of some of my personal pixel-art creatures.
![A small pixel-art insect.](https://github.com/SpiffyJohnson/Pixel-Monster-Bestiary/blob/main/img/monsters/sprite_21.png)


<h2 align="center">Features</h2>

The Images page
------

In this page, the user can view all of the monster images in the site in one place, side-by-side.

<img width="1836" height="951" alt="image" src="https://github.com/user-attachments/assets/825ead96-716f-42cd-8c54-e70d266af4dc" />

The Sounds page
------

On this web page, the user can flip between all of the available monsters. Clicking the sound button on this page will play the current monster's roar.

<img width="1836" height="881" alt="image" src="https://github.com/user-attachments/assets/bc947c88-d4a2-4c40-8edb-77a02bc62418" />

The Book Page
------

By flipping through the pages of the Monster Book, users can learn more about the monsters detailed within.
